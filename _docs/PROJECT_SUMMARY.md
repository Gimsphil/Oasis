## ✨ 핵심 기능

### 1️⃣ **자료사전 연동** (TAB)
- 자료사전 자동 팝업
- 데이터 자동 전송
- 코드 자동 입력

### 2️⃣ **1식 저장** (ESC)
- 합계 자동 계산
- 부모 테이블 동기화
- 팝업 자동 종료

### 3️⃣ **실시간 피드백**
- 합계 실시간 표시
- 색상 자동 강조
- 유효성 자동 검사

### 4️⃣ **빠른 조작**
- Ctrl+S: 저장
- Ctrl+Z: 복구
- F9: 조각파일

### 5️⃣ **직관적 입력**
- 자동 완성
- 드래그 드롭
- 검색 자동화
